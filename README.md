# Blackjack Trainer

Built for Card Counting & Quantitative Finance II.

A little browser game to practice basic strategy and card counting (Hi-Lo).

The idea is simple: blackjack is a fun, low-stakes way to get comfortable with probability, expected value, and decision-making under uncertainty the same stuff that shows up later in quant finance, just with cards instead of markets.

## What it does

- Play blackjack against a computer dealer (choose 4, 6, or 8 decks)
- Basic Strategy mode: tells you if your hit/stand/double/split/surrender call was correct
- Card Counting mode: shows a live running count and true count (with toggles to hide them so you can practice tracking on your own)
- Wallet Function: practice with a €10,000 starting bankroll and €50 minimum bet, so you can see how strategy actually affects your money over time
- Hand history and performance chart so you can see how you're doing over a shoe
- A few color themes because why not
