# Blackjack Counting Course

A self-contained course on blackjack basic strategy and card counting — covering
practice, reference material, and the math behind the game.

## Structure

| Folder | Contents |
|---|---|
| `practice/` | Interactive practice tool (`index.html`) |
| `presentations/` | Lecture slides on card counting and quantitative finance |
| `sheets/` | Reference sheets — basic strategy, table rules, session log |
| `simulations/` | Python scripts for house edge, EV, and bankroll risk |

## Presentations

- *Card Counting & Quantitative Finance I*
- *Card Counting & Quantitative Finance II*
- *Card Counting & Quantitative Finance III*

## Reference sheets

Available as `.docx` and `.pdf`:
- **Basic Strategy Sheet**
- **Rules Sheet**
- **Log Sheet**

## Simulations

- `blackjack_sim.py` — house-edge and EV simulator (basic strategy, Hi-Lo, Omega II)
- `bankroll_bot_sim.py` — bankroll and risk-of-ruin simulator
